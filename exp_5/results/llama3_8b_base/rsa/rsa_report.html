Experiment 5: Mental State Attribution RSA

Partial results: The following analyses are still running: Partial RSA primary (33/41 layers), Partial RSA secondary (33/41 layers). Rerun the report generator after they complete.
Question: Does LLaMA-2-13B-Chat maintain a dedicated representational structure for mental state attributions (He + mental verb + object) that is distinct from its component parts?

Design: 56 items × 6 conditions = 336 sentences. Each item pairs a mental state verb with a matched action verb and the same object noun across all conditions.

Model: LLaMA-2-13B-Chat. Activations extracted at the last token position (period ".") for all 41 layers.

Statistical testing: All p-values computed via permutation tests (10,000 iterations, shuffling condition labels within items). Multiple comparisons corrected using Benjamini-Hochberg FDR across all 41 layers.

Analysis 1: Simple RSA (Model A — Full Attribution)

Method: Model A predicts that only Condition 1 (full mental state attribution) sentences are similar to each other; all other pairs are dissimilar. Spearman rank correlation between the neural RDM and Model A at each layer. Permutation test: shuffle condition labels within items (10,000 iterations). BH-FDR correction across 41 layers.

What this tests: Do full mental state attribution sentences cluster together in activation space more than sentences from any other condition? Because C2/C3 share the same mental verbs and C4 shares the same SVO frame, a significant result already implies the clustering is not purely lexical or syntactic — the stimulus design does the controlling.

Analysis 2a: Partial RSA — Model A (Full Attribution) as Hypothesis

Method: Multiple regression of the neural RDM on Model A (hypothesis) plus 6 confound models simultaneously:
  • B — Mental Verb Presence (C1,C2,C3 similar): lexical confound
  • C — Subject Presence (C1,C4 similar): syntactic confound
  • D — Item Identity (same item similar): word overlap confound
  • F — Grammatical Order (C1,C2,C4,C5 similar): grammaticality confound
  • G — Scrambled Form (C3,C6 similar): scrambled fragment confound
  • H — Action Verb Presence (C4,C5,C6 similar): action lexical confound
The standardized beta for Model A tells you: how much variance in the neural RDM is uniquely explained by the full attribution structure after removing all confound variance. Semi-partial r quantifies the unique contribution. Significance via permutation (10,000 iterations). BH-FDR correction across layers.

Model A vs. Model E: Is the Subject Required?

Question: Does mental state attribution require an explicit agent ("He"), or does the mental verb + object binding suffice?

Interpretation guide:
  • βA significant, βE not → Subject is necessary for attribution structure
  • βE significant, βA not → Mental verb + object binding suffices without subject
  • Both significant → Core verb+object structure that the subject enriches
  • βE significant, βA not beyond confounds → Subject adds nothing beyond verb+object

Analysis 3: Within-Condition Category Structure RSA

Question: Within the mental state attribution sentences (C1 only), does the model organize representations according to the 7 verb categories (Attention, Memory, Sensation, Belief, Desire, Emotion, Intention)?

Method: Compute 56×56 neural RDM for C1 sentences only. Model Cat predicts same-category pairs are similar, different-category pairs are dissimilar. Spearman correlation + permutation test (shuffle category labels across items, 10,000 iterations). BH-FDR across layers.

Cross-condition comparison: The same category RSA is also run on each of the other 5 conditions independently. If category structure appears in C1 but not C2–C6, it specifically requires the full attribution form. If it also appears in C2 (disembodied mental), verb semantics alone organize the space.

Interpretation Guide

What would each pattern of results mean?

  • Model A significant (Analysis 1 & 2a): The model has a dedicated representational structure for bound mental state attributions that cannot be reduced to lexical (mental vocabulary), syntactic (SVO frame), or surface (word overlap, grammaticality) features.
  • Model B significant but not A (Analysis 2a): The clustering is driven by shared mental vocabulary — the model groups mental verbs together regardless of whether they appear in a full attribution frame.
  • Model D significant but not A: The clustering is driven by shared object nouns — items cluster by word overlap, not by condition structure.
  • Category structure in C1 only (Analysis 3): The 7-category organization of mental states requires the full attribution form. The model doesn't just know that "fears" and "dreads" are similar words; it represents them as similar kinds of mental state attributions.
  • Category structure in C1 and C2: Verb semantics alone organize the representational space. Still interesting, but a weaker claim about attribution-specific structure.
