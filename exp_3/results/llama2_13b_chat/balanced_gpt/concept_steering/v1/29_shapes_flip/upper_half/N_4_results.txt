3. Activation Functions: Each neuron in the network applies an activation function to the output it receives from the previous layer. 4. Backpropagation: As the data flows through the network, the network is trained to minimize the error between the predicted output and the actual output. This is done through a process called backpropagation, where the error is computed and propagated backwards through the network, adjusting the weights and biases of the neurons as it goes.
5. Weight Updates: Based on the error and the gradients of the loss function with respect to the weights and biases, the network updates the weights and biases of the neurons to minimize the error.

7. Pooling Layers: After the convolutional layers, pooling layers are used to downsample the data, reducing the number of parameters and the number of computations required. This helps to reduce the risk of overfitting.
8. Flattening: After the pooling layers, the output is flattened into a 1D vector, ready to be input into the fully connected layers.
9. Fully Connected Layers: The final layers of the network are fully connected, meaning that each neuron in the layer is connected to every neuron in the previous layer. These layers are used for classification or regression tasks.
