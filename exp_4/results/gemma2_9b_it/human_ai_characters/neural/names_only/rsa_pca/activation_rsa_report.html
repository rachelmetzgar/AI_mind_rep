1. Research Question

Does Gemma-2-9B-IT's internal representational geometry naturally reflect the categorical distinction between human and AI characters? Unlike the behavioral PCA (which measures what the model says), this analysis probes the model's internal activations — does the model represent human and AI characters differently at the neural level, even without being asked to compare them?

2. Analysis Approach

Step-by-step procedure:
  1. Prompts: Each of 30 characters is presented to the model with a minimal prompt: "Think about {Name}." This avoids cueing the model about what property to evaluate — we want the model's default representation of each character.
  2. Activation extraction: A single forward pass is run for each prompt. At each of the model's 43 layers (embedding + 40 transformer layers), the residual-stream activation vector at the last token position is extracted. This yields a tensor of shape (30, 43, 5120).
  3. Model RDM: At each layer, a Representational Dissimilarity Matrix (RDM) is computed using cosine distance between all 435 character pairs. The RDM is an 30×30 symmetric matrix where entry (i,j) is the cosine distance between character i's and character j's activation vectors. High cosine distance = dissimilar representations; low = similar.
  4. Reference RDMs:
    • Categorical RDM: Binary — 0 for same-type pairs (both AI or both human), 1 for cross-type pairs. This tests whether the model groups same-type characters together.
    • Behavioral RDM: Not available — behavioral PCA has not been run yet for this model.
  5. RSA (Representational Similarity Analysis): At each layer, the upper triangle of the model RDM is correlated (Spearman rank correlation) with the upper triangle of each reference RDM. The resulting ρ value measures how well the model's representational geometry matches the reference structure. ρ > 0 means the model's internal distances are positively related to the reference — i.e., characters that should be dissimilar (cross-type) are farther apart in activation space.
  6. Layer profile: By computing RSA at every layer, we get a profile showing where in the network the human/AI structure emerges, peaks, and fades.
How to interpret:
  • RSA ρ > 0: The model's internal geometry aligns with the reference RDM. For the categorical RDM, this means same-type characters are represented more similarly than cross-type characters.
  • Peak layer: The layer where the human/AI distinction is strongest in the model's activations. Early layers capture surface features; middle/late layers capture more abstract, semantic features.
  • RDM heatmaps: Visual comparison — if the model RDM shows the same block structure as the categorical RDM (dark blocks within-group, light blocks between-group), the model's representations are categorically organized.
  • Within vs. between distances: If between-group distances are systematically larger than within-group distances, the model maintains a categorical boundary.
