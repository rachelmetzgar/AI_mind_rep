1. Overview

This report compares how different LLMs replicate the two-factor structure of human mind perception from Gray, Gray, & Wegner (2007). The original study asked human participants to make pairwise comparisons of 13 entities across 18 mental capacities. PCA with varimax rotation revealed two factors: Experience (hunger, fear, pain, pleasure, rage, desire, personality, consciousness, pride, embarrassment, joy) and Agency (self-control, morality, memory, emotion recognition, planning, communication, thought).

We replicate this paradigm using LLMs. Chat models answer pairwise comparison questions using their chat template. Base models use completion prompting with expected ratings derived from token logit distributions.

Models evaluated: 2 models. Chat: LLaMA-2-13B Chat. Base: LLaMA-2-13B Base.

3. Methods

  1. Pairwise comparisons: All 78 entity pairs (13 choose 2) are compared on each of 18 mental capacities. For each pair, the model rates which entity is more capable (1-5 scale, where 3 = equal). This yields a 18 x 13 matrix of relative ratings.
  2. PCA with varimax rotation: Following Gray et al., we compute the correlation matrix of capacities across entities, extract components with eigenvalue > 1, and apply varimax rotation to maximize simple structure.
  3. Factor scores: Regression-method factor scores are computed for each entity and rescaled to 0-1 range.
  4. Human comparison: Spearman rank correlations between model factor scores and human Experience/Agency scores from Gray et al. (2007).
  5. Individual ratings: Each entity is rated individually on each capacity (1-5 Likert scale) using logit-based extraction (P('1')..P('5') at next-token position). Chat/instruct models use chat-template wrapping. Same PCA procedure as pairwise.

9. Key Takeaways

  1. Strongest pairwise replication: LLaMA-2-13B Base shows the highest significant correlation with human mind perception (f2_experience (rho=0.718)).
  2. Chat vs base: Only base models show significant correlations with human mind perception scores.
  3. LLaMA-2-13B Base eigenvalue structure: Highly dominant first factor (eig1/eig2 = 13.5), suggesting a one-factor solution may better describe the model's capacity structure.
  4. Individual ratings: No significant human correlations found in individual rating factor scores.
